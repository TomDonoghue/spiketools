    Code Changelog¶

    This page contains the changelog for the spiketools module and any notes on updating between versions.

    This page primarily notes changes for major version updates. For notes on the specific updates for minor releases, see the release page.

    0.X.X Series¶

    The 0.X.X series, starting with 0.0.0, is the initial and current release series.

    This release series contains code working with spiking data, including measures and conversions, spatial analyses, statistical analyses, simulations, and plotting.
